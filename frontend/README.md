# Frontend

Клиентская часть тестового задания Avito: список объявлений, просмотр, редактирование и AI-подсказки.

## Запуск

```bash
cd frontend
npm i
npm run dev
```

## Env (опционально)

`frontend/.env`

```env
VITE_API_URL=http://localhost:8080
```

## Реализовано по ТЗ

- `/ads`: список, поиск, сортировка, фильтры, пагинация, loading/error
- `/ads/:id`: карточка объявления, характеристики, блок недостающих полей, loading/error
- `/ads/:id/edit`: форма с валидацией, поля по категории, AI-кнопки, сохранение/отмена, loading/error
- AI: единый ответ `message`, применение ответа в форму
- Черновик формы в `localStorage` с восстановлением после перезагрузки
- Отмена HTTP-запросов при переходах между страницами

## Важные решения

- Для чекбокса использован `@headlessui/react` из-за проблемы в нужном сценарии с Mantine
- На `/ads/:id` добавлена стрелка назад к списку (в макете не было)
